Overview

Wadah Pecel AE is an Android application owned by BPS (Badan Pusat Statistik) which contains data belonging to BPS for user to access. This application is a development project during practical work. Development has been carried out for this Wadah Pecel AE application.

Previously, Wadah Pecel AE application was a static application, where to update the data you needed to update the application to the latest version. Meanwhile, for now, the application has been developed to be more dynamic, namely by integrating with Firebase so that to update data there is no need to update the application again. Coding is no longer needed to update data which can be done in Firebase. Apart from that, there are updates to the application such as added navigation, search features, update dialog alerts, and several displays that have become more responsive.
