Overview

Semar (Sewa Kamar) is an Android-based room registration application in Undip flats. This application was created for a final project. This application is designed to make it easier for student to register rooms and admin to manage the administration system in the flat.

In this application there are 2 roles, namely admin and user who will have a dashboard and access to each role. The features that can be provided from the application are authentication features, room and building info, room registration features, payment features, notification features, order history features, profile edit features, helpdesk features, and data CRUD features for admins. This application is also connected to a MySQL database for managing student and room data.
