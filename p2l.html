Overview

P2L (Pekarangan Pangan Lestari) is a place in Gedongan village to manage various plants. There is a P2L website that aims to provide information about P2L so that P2L can be known by the public. This website was created for a project during KKN.

This website has two roles, namely admin and user. There is an authentication feature for the admin. On the dashboard, admin can CRUD any data and information on the website. User can see all the information on the website. There is information about plants and activities at P2L. All plant and activity data is connected to the admin dashboard and MySQL database, making it easier for admins to manage the website.
